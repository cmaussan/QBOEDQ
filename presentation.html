SPORE

SPecifications for a POrtable Rest Environment

"SPORE is a specification for describing REST API that can be parsed and used automatically by clients implementations to communicate with the described API."

  • Créé par Franck Cuny et Nils Grünwald au sein de Linkfluence.
  • Première release il y a 1 an à l'OSDC.fr 2010 :)

Pourquoi ?

Chez Linkfluence, on développe des moteurs de captation/traitement de données issues du web social.

  • Système distribué d'agents dans plusieurs langages
  • Communications avec bases, outils externes, etc. en REST
  • Write once, deploy everywhere

Perl

 1 use Net::HTTP::Spore;
 2 use Try::Tiny;
 3 
 4 my $client = Net::HTTP::Spore->new(
 5     '/path/to/spec.json',
 6     base_url => 'http://base_url/', # Optional
 7 );
 8 $client->enable( 'Format::JSON' );
 9 
10 my $res;
11 try {
12     $res = $client->search( format => 'json', q => 'osdc' );
13 }
14 catch {
15     warn $_;
16 };
17 
18 say $_->{ text } for( @{ $res->body->{ results } } );
  • Client complet (en prod).
  • Développé chez Linkfluence par Franck Cuny.
  • Net-HTTP-Spore est dispo sur CPAN et sur GitHub.

Lua

 1 local Spore = require 'Spore'

Ruby

 1 require 'spore'
 2 require 'spore/middleware/format'
 3 
 4 client = Spore.new(
 5     '/path/to/spec.json',
 6     { :base_url => 'http://base_url/' } 
 7 )
 8 client.enable(Spore::Middleware::Format)
 9 
10 res = begin
11     client.search( :format => 'json', :q => 'osdc' )
12 rescue
13     raise $!
14 end
15 
16 res.body.results.each do |t|
17     puts t.text
18 end
  • Client complet (en prod).
  • Développé chez Weborama par Sukria.
  • Ruby-Spore est dispo sur GitHub.

nodejs

 1 var spore = require( 'spore' );
 2 var client = spore.createClient( __dirname + '/path/to/spec.json' );
 3 client.enable( spore.middlewares.json() );
 4 
 5 client.search( { format: 'json', q: 'osdc' }, function( err, res ) {
 6     if ( err ) 
 7         throw err;
 8     for( i in res.body.results ){
 9         console.log( res.body.results[i].text );
10     };

Clojure

 1 (import 'clj-spore)
 2 (import 'clj-spore.middleware)
 3 
 4 (def client (load-spec-from-file "/path/to/spec.json"
 5                              :middlewares [wrap-json-format]
 6                              :overload {:base_url "http://base_url/"}))
 7 (try
 8     (let [res ((client :search) :format "json" :q "osdc")]
 9         (doseq [item (:decoded-body res)] (println (item :text))))
10     (catch Exception e (println (.getMessage e))))
  • Client complet (en prod)
  • Développé chez Linkfluence par Nils Grünwald.
  • clj-spore est dispo sur GitHub
